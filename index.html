Installation

Note: Please make sure you have installed node and npm in your system. You can check your node version and npm version by using the following command:

node --version
npm --version

Follow the steps below to setup your site template:

  1. Install yarn Globally and, verify the installed version:

    2. npm install --global yarn  
    yarn --version
  2. Move into your project directory and change a node module into 16.13.2 using nvm 

    3. nvm install 16.14.2 
    nvm list
    nvm use 16.14.2
  3. install Node Module

    4. yarn
  4. Run the App

    5. yarn start
  5. Open the /src folder to find all the template files. You will need to upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  6. Below is the folder structure and needs to be uploaded to your website or localhost root directory:
    • src/assets - Contains all of the assets referenced
      • src/assets/css - base and loader files
      • src/assets/images - base and logo images files
      • src/assets/svg - AllSvg icons files
    • src/page - Contains all Components
    • src/index.css - Main css File
    • public/index.html - Homepage

    Theme Customization

    A theme is a collection of color and typography options. Each theme includes three palettes that determine component colors:

  • A primary palette for the color that appears most frequently throughout your application
  • An accent, or secondary, palette used to selectively highlight key parts of your UI
  • A warn, or error, palette used for warnings and error states

    • You can include the CSS styles for a theme in your application in one of two ways: by defining a custom theme with Sass, or by importing a pre-built theme CSS file.

    Customizing Colors

    when you do need to customize your palette, you can configure your colors under the colors key in the theme section of your tailwind.config.js file:

    If you’d like to completely replace the default color palette with your own custom colors, add your colors directly under the theme.colors section of your configuration file:
